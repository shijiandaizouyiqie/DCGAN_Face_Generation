### 预处理

由于数据集较大（20w张图片），所以直接在训练时再将数据导入Dataloader中会出现CPU瓶颈的现象。即CPU占用率特别高，GPU占用率却偏低。

所以建议下载好数据集之后，先使用 `img_resize_to_64x64.py`提前将数据集裁剪为64x64大小的图片保存到新文件夹中。这样会加快一定训练速度。
